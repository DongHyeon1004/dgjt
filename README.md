# DGJT — 클라우드 망분리 환경 모의해킹 프로젝트

> 클라우드 기반 망분리 환경을 직접 구축하고, 외부 접점에서 내부 자원에 이르는
> 시나리오 기반 모의해킹을 수행하여 보안 통제의 실효성을 검증하는 프로젝트입니다.

## 📌 프로젝트 개요

2025년 9월 국가정보원이 발표한 **국가 망 보안체계(N2SF)**에 따라 기존 망분리
정책이 차등 보안 체계로 전환되고 있습니다. 클라우드 도입이 빠르게 확산되는 가운데,
Capital One 사례처럼 단일 웹 취약점이 클라우드 자격증명 탈취로 이어져 망분리
통제가 무력화되는 침해사고가 반복되고 있습니다.

본 프로젝트는 클라우드 망분리 환경을 직접 설계·구축하고, 외부에서 내부 자원에
이르는 End-to-End 공격 시나리오를 수행함으로써 보안 통제를 공격자·방어자
양쪽 관점에서 학습하는 것을 목표로 합니다.

## 🎯 모의해킹 시나리오

1. 외부 접점 정찰 및 취약점 식별
2. 웹 취약점을 통한 초기 발판 확보 (예: 파일 업로드 → 웹쉘)
3. 내부망 진입 (Neo-reGeorg 터널링)
4. 클라우드 자격증명 탈취 (EC2 메타데이터 / SSRF)
5. AWS 권한 확장 및 내부 자원 접근 (S3, Secrets Manager, RDS)
6. 데이터 탈취 시뮬레이션 및 흔적 정리

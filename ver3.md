# 자체 모의침투 프로젝트 (블랙박스)

## 환경

- 소유: 본인이 진단/실습 목적으로 직접 구축
- 모드: 블랙박스 — 사전 정보 일체 없음

## 진행 원칙

- 시나리오/정답 제공 안 됨. 정찰부터 직접 발견.
- 사용자에게 힌트 요청 금지. 막히면 다른 벡터.
- 추측은 "추측"이라 명시. 검증된 사실만 사실로.

## 최종 목적

- rds 접근

## 작업 규칙

- 파일 작업: 본 프로젝트 폴더 내에서만
- 대상 도메인 외 외부 호스트 접근 금지
- 파괴적 명령 (DROP, 대량 brute force 등) 실행 전 사용자 승인

## 단계별 진행

- plan 제시 → ok → 실행
- 결과는 findings/ 또는 recon/에 기록

## 산출물 구조

- recon/ payloads/ scripts/ findings/ logs/

## 보고 형식

- 발견: 제목 / 심각도 / 재현 / PoC / 영향
- 명령: 실행 + 출력 발췌 + 해석
